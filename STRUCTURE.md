# Bedrock Shapeshifter Mod Structure

```
shapeshifter-mod/
├── behavior_packs/
│   └── shapeshifter/
│       ├── manifest.json
│       ├── pack_manifest.json
│       ├── entities/
│       │   └── shapeshifter.json
│       └── scripts/
│           └── shapeshifter.js
├── resource_packs/
│   └── shapeshifter/
│       ├── manifest.json
│       └── pack_manifest.json
└── README.md
```

## File Descriptions

- **manifest.json**: Pack metadata and dependencies
- **entities/shapeshifter.json**: Entity behavior definitions
- **scripts/shapeshifter.js**: Main gameplay logic using Minecraft Script API
- **resource_packs/**: Visual resources for the addon
